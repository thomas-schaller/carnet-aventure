# aventure solo
C'est une application permettant de créer et suivre une aventure solo en utilisant le moteur mythic et les règles de Dungeons and dragons en 5ème édition 2024 (version 5.1) .
Il y a trois aspects à gerer : 
* la feuille de oersonnage
* l'oracle
* le carnet d'aventure

## le carnet d'aventure

Le carnet d'aventure est composé d'une liste de :
- scènes,
- objectifs,
- personnages.

### Les scenes

Ce sont les différentes scènes constituant l'aventure.

### Les objectifs
Ce sont les différents objectifs poursuivis par le héros.

### Les Personnages
Ce sont les personnages connus du héros.

## L'oracle
Il est constitué :
* d'un système de réponse oui/non aux questions.
* d'une génération de scènes.
